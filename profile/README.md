## Contributing
We are always looking for help on our [Projects](https://github.com/orgs/code-org-open-source/repositories). Here are some ways you can contribute:

- Opening Issues  
Found an Bug or Security risk in our [Project](https://github.com/orgs/code-org-open-source/repositories)? Open a Issue so we can get it fixed! üòÅ  
- Contributing Code  
Want to help us by Coding? Open a [pull-request](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests/creating-a-pull-request), and we will give it a look!   
- Documentation   
Know everything about how our [Project](https://github.com/orgs/code-org-open-source/repositories) works? Write some documentation to help other people know how it works! üìñ
